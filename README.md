# alink

## discription :
    This is a small open source tool for creating aliases(nicknames) for other open source applications

## usage:
  ### *to display help* :
     bash alink.sh 
  ### *first method* :
    bash alink.sh [APPLICATION] [NICKNAME]
  ### *second method* :
    bash alink.sh -i 
    alink [APPLICATION] [NICKNAME]
